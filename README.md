# 📊 Customer Segmentation & Ad Performance Analysis using Clustering & PCA

## 📌 Project Overview
Built an advanced customer segmentation model to analyze ad performance and uncover high-value segments using unsupervised machine learning techniques. The project focuses on improving marketing efficiency and maximizing ROI through data-driven insights.

---

## 🎯 Objectives
- Segment ads/customers based on performance metrics (CTR, CPC, CPM)  
- Identify high-performing and underperforming segments  
- Optimize marketing strategies and budget allocation  

---

## 📊 Dataset
- 23,000+ records with multiple ad performance features  
- Includes metrics like impressions, clicks, spend, CTR, CPC, CPM  

---

## ⚙️ Approach
- Performed data cleaning and missing value imputation using domain formulas  
- Removed outliers using IQR to improve clustering performance  
- Applied Z-score normalization for feature scaling  
- Conducted detailed Exploratory Data Analysis (EDA)  

---

## 🤖 Modeling Techniques
- Hierarchical Clustering (Dendrogram analysis)  
- K-Means Clustering (Elbow Method for optimal clusters)  
- Silhouette Score Evaluation (**0.64**)  
- Principal Component Analysis (PCA) for dimensionality reduction  

---

## 📈 Key Results
- Identified 3–4 distinct customer/ad segments  
- Reduced dimensionality from 60 → 5 features (90% variance retained)  
- Achieved well-separated and interpretable clusters  

---

## 🔍 Key Insights
- High CTR + Low CPC segments deliver maximum ROI  
- Certain segments require increased ad investment for better returns  
- Poor-performing segments can be optimized through targeted strategies  

---

## 💡 Business Impact
- Enables targeted marketing and personalization  
- Improves ad spend efficiency and ROI  
- Supports data-driven campaign optimization  

---

## 🛠️ Tech Stack
Python | Pandas | NumPy | Scikit-learn | Matplotlib  

---

## 📌 Conclusion
The project successfully identified actionable customer segments and demonstrated how clustering and PCA can be leveraged to improve marketing performance and strategic decision-making.
